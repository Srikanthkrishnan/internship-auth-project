# Internship Authentication Project

## Technologies Used
- PHP
- MySQL
- MongoDB
- Redis/Memurai
- Bootstrap
- AJAX

## Features
- User Registration
- User Login
- Profile Management
- Redis Session Storage
- MongoDB Profile Storage

## Run Project

1. Start XAMPP
2. Start MongoDB
3. Start Memurai
4. Open:
http://localhost:8080/internship-project
